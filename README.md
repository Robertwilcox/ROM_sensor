# ROM_sensor
ROM sensor for data-driven workouts

## System and Software Requirements
The project was built on the Boolean board and tested on `Windows 10` with the 
following tools:
* Xilinx Vitis IDE 2022.2
* Xilinx Vivado 2022.2
* PuTTY
